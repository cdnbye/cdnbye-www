使用方法

SwarmCloud

# 准备工作

# 仅iOS平台

如果只针对iOS平台，可以直接用 CocoaPods 安装SDK，编辑 Podfile

target 'TargetName' do
# Uncomment the next line if you're using Swift
# use_frameworks!
    pod 'SwarmCloudSDK', '= 2.6.5'
end

然后，运行如下的命令：

$ pod install

如果提示找不到SwarmCloudSDK，执行以下命令：

$ pod repo update

如果需要更新SDK到最新版本，执行：

pod update SwarmCloudSDK --verbose --no-repo-update

# iOS/tvOS/macOS 平台

如果需要用于tvOS或macOS，则需要先安装依赖，然后将两个 framework 拷贝到xcode中。

# 安装依赖

# CocoaPods

编辑 Podfile

target 'TargetName' do
# Uncomment the next line if you're using Swift
# use_frameworks!
    pod 'SocketRocket', '~> 0.6'
    pod 'CocoaLumberjack', '~> 3.7'
    pod "GCDWebServer", :git => 'https://github.com/swisspol/GCDWebServer.git'
    pod 'CocoaAsyncSocket', '~> 7.6'
    pod 'PINCache', '~> 3.0'
end

然后，运行如下的命令：

$ pod install

# Carthage

编辑 Cartfile:

github "facebook/SocketRocket" ~> 0.6
github "CocoaLumberjack/CocoaLumberjack" ~> 3.7
github "pinterest/PINCache" ~> 3.0
github "https://github.com/swisspol/GCDWebServer.git" "master"
github "robbiehanson/CocoaAsyncSocket" ~> 7.6

然后，运行如下的命令：

carthage update --use-xcframeworks --cache-builds

这个命令会在 Carthage/Build 文件夹中生成编译好的XCFramework。
依次打开xcode的 General -> Frameworks, Libraries, and Embedded Content，将所有 XCFramework 拖进来。

# 集成 SwarmCloudSDK 和 WebRTC 框架

# 下载预编译好的 XCFramework

  • SwarmCloudSDK.xcframework (opens new window)
  • WebRTC.xcframework (opens new window)

# 拷贝 framework

先解压这2个文件，然后将 SwarmCloudSDK.xcframework 和 WebRTC.xcframework 拖入 General -> Frameworks, Libraries, and Embedded Content

# 集成方法

SDK通过本地代理伺服器拦截数据请求的方式来进行P2P缓存和传输，所以需要在项目的info.plist中允许HTTP请求：

<key>NSAppTransportSecurity</key>
<dict>
    <key>NSAllowsArbitraryLoads</key>
    <true/>
</dict>

# 引入 SwarmCloudSDK

在工程的 AppDelegate.m 文件导入头文件：

import SwarmCloudSDK

#import <SwarmCloudSDK/SwarmCloudSDK.h>

# 初始化 SWCP2pEngine

在工程 AppDelegate.m 的 application:didFinishLaunchingWithOptions: 方法中初始化：

func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
    let config = SWCP2pConfig.defaultConfiguration()
    config.announceLocation = SWCAnnounceLocation.china      // Set HongKong or USA if you changed zone
    SWCP2pEngine.sharedInstance().start(token: YOUR_TOKEN, p2pConfig: config)
    return true
}

- (BOOL)application:(UIApplication *)application didFinishLaunchingWithOptions:(NSDictionary *)launchOptions
{
    SWCP2pConfig *config = [SWCP2pConfig defaultConfiguration];
    config.announceLocation = SWCAnnounceLocationChina;    // Set HongKong or USA if you changed zone
    [[SWCP2pEngine sharedInstance] startWithToken:YOUR_TOKEN andP2pConfig:config];
    return YES;
}

其中 YOUR_TOKEN 是用于标识用户的字符串，请换成自己的token，点击这里查看如何注册 Appid 并获取 token。

# 转换地址

在代码中实例化AVPlayer之后（也可以是其他任何视频播放器），先将URL传给 SWCP2pEngine，之后将转化的本地URL传给播放器：

let orginalUrl = URL.init(string: "https://your_stream.m3u8")
let parsedUrl = SWCP2pEngine.sharedInstance().parse(streamURL: orginalUrl)
_player = AVPlayer.init(url: parsedUrl)

NSURL *originalUrl = [NSURL URLWithString:@"https://your_stream.m3u8"];
NSURL *parsedUrl = [[SWCP2pEngine sharedInstance] parseStreamURL:originalUrl];
_player = [[AVPlayer alloc] initWithURL:parsedUrl];

就这么简单，你的播放器已经具备P2P能力了！

由于模拟器运行环境问题，请在真机进行调试。
